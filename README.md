## Description
GREH is a full-stack property listing web application built with **Node.js, Express, MongoDB, and EJS**. It provides a seamless platform for users to browse, add, and review property listings. The project emphasizes clean schema design, smooth CRUD operations, and an intuitive user experience.  
Currently, the app is in development, with ongoing work on **review integration, schema validation, and responsive layouts**.

## Features
- Add, display, and delete property listings
- User reviews linked to listings (integration in progress)
- Schema validation for data integrity
- Responsive design for desktop and mobile
- Secure backend with MongoDB
- Error handling and route management

## Tech Stack
- **Frontend:** EJS, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
